Proyecto de Maestría

Durante mi estancia desarrolle un proyecto de software para una empresa dedicada a proveer soluciones de comunicación, mi proyecto consta del desarrollo de una aplicación web SPA, utilizada para la gestión administrativa de uno de sus servicios, la tarea principal de esta aplicación es la de notificar automáticamente a los clientes que su periodo de pago esta por expirar.

Arquitectura de la herramienta propuesta

La herramienta propuesta es una solución que consta de tres proyectos relacionados entre sí, esto para generar capas de abstracción entre los datos, entidades y las reglas de negocio del software.

Modelos del sistema desarrollados en la herramienta de Enterprise Architect

Los requerimientos del sistema fueron modelados en una herramienta de lenguaje de presentación grafica descriptiva (UML), para dicha tarea se utilizó la herramienta de Enterprise Architect 12.0, los modelos que se desarrollaron son los que podemos visualizar en la imagen de un costado.

Tecnologías utilizadas en el desarrollo de la aplicación web

Para el back-end de la aplicación se utilizo .NET framework con la tecnología de ASP .NET Core utilizando SQL Server como gestor de base de datos.
Para nuestro front-end se utilizo el framework de Vuetify y las tecnologías de HTML5, CSS y JS.
